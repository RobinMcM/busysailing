# Avatar Models Directory

## Place your GLB avatar files here

This directory should contain:
- `consultant.glb` - Female consultant avatar
- `partner.glb` - Female partner avatar

## How to Get Free Avatars

See the main `AVATAR_SETUP.md` file in the project root for detailed instructions on getting free realistic 3D avatars from Ready Player Me.

## What Happens Without GLB Files

If the GLB files are not present, the system will automatically fall back to displaying professional stock photos (already included in the project).
